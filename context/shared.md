# Shared Organisation Context

This context is injected into every agent in the pipeline.
Use it only for project or organisation facts you want every agent to reference.

## What belongs here

- Organisation or product name
- Current strategic priorities / OKRs
- Shared user segments or personas
- Fixed constraints such as compliance, platform, contractual, or staffing limits
- Canonical terms the team uses
